# Q-Learning Snake

## 项目简介

这是一个使用Q-learning强化学习算法训练的贪吃蛇游戏。智能体通过10000轮训练，学会了如何在游戏环境中高效地收集食物并避免碰撞。

### 核心特性
- 使用Q-learning算法进行强化学习
- 优化的状态空间设计（5维特征）
- 完整的游戏界面（开始屏幕、游戏画面、结束屏幕）
- 详细的训练数据和结果分析
- 训练后的模型保存和加载功能

## 安装和运行

### 环境要求
- Python 3.12+
- pygame
- numpy
- matplotlib

### 安装依赖
```bash
# 创建虚拟环境（可选）
python -m venv venv

# 激活虚拟环境
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate

# 安装依赖
pip install pygame numpy matplotlib
```

### 运行项目
```bash
# 运行训练和演示
python qlearning贪吃蛇优化版.py

# 训练完成后，会自动启动游戏演示
```

## 训练结果

### 训练配置
- **总训练轮数**: 10,000
- **训练时间**: 约9秒
- **状态空间大小**: 60个状态
- **Q-learning参数**:
  - 学习率 (ALPHA): 0.2
  - 折扣因子 (GAMMA): 0.95
  - 初始探索率 (EPSILON): 0.9
  - 探索率衰减 (EPS_DECAY): 0.999
  - 最小探索率 (MIN_EPS): 0.05

### 游戏控制
- **开始游戏**: 按空格键

## 技术实现

### 状态空间设计
智能体使用5维特征表示游戏状态：
1. 食物相对方向（0-上，1-右，2-下，3-左）
2. 危险上（1=危险，0=安全）
3. 危险右
4. 危险下
5. 危险左

### 奖励函数
- 吃到食物: +100
- 向食物移动: +1
- 远离食物: -1
- 碰撞墙壁/自身: -50
- 超时: -10

### UI界面
- **开始屏幕**: 游戏标题、开始提示、训练信息
- **游戏界面**: 蛇身、食物、分数、奖励
- **结束屏幕**: 游戏结束提示、最终分数、总奖励
